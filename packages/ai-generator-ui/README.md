# livepeer-ai-generator
This repo demonstrates how to use the Livepeer AI Gateways (Livepeer.Cloud's Gateway).

Make sure you are running Node 20 (or higher)

## Install Prereqs

`npm install`

## Run for Dev

`npm run dev`

## Run the Build

`npm run build`


## Test the UI

Open your browser to `http://localhost:5173/`
